# Simply Plot

Pet project to showcase R Studio plotting.

## Instructions

- clone the repository.
- checkout a new feature branch 

`git checkout -b feature/my-new-plot`

- Add any plot file ending in `.R`

`git add *`
`git commit -m "my new plot"`
`git push origin feature/my-new-plot`

- Create the pull request, 
this 
- triggers the workflow 
- creates a zip file of all the plots and saves as an artifact
- the artifact is available to download as a zip file containing any plots.
- download and unzip, you'll see the plots as png files.
